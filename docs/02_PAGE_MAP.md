# 页面地图

## 1. 公共页面

- `/`：首页
- `/schools`：学校列表
- `/schools/[schoolId]`：学校详情
- `/programs`：专业 / 项目列表
- `/programs/[programId]`：项目详情
- `/resources`：资源库
- `/guides`：签证、租房和生活指南

## 2. 认证页面

- `/login`：登录
- `/register`：注册
- `/forgot-password`：找回密码

## 3. 学生工作台

- `/dashboard`：学生首页概览
- `/profile`：学生背景档案
- `/recommendations`：选校推荐
- `/applications`：申请清单
- `/applications/[applicationId]`：单个申请项目详情
- `/timeline`：申请时间线
- `/tasks`：任务管理
- `/documents`：文书与简历中心
- `/recommendation-letters`：推荐信管理
- `/offers`：Offer 列表
- `/offers/compare`：Offer 对比

## 4. 社区与 Mentor 页面

- `/community`：社区首页
- `/community/posts/[postId]`：经验帖详情
- `/mentors`：学长学姐 / Mentor 列表
- `/mentors/[mentorId]`：Mentor 主页

## 5. 后台管理页面

- `/admin`：后台首页
- `/admin/schools`：学校管理
- `/admin/programs`：项目管理
- `/admin/deadlines`：DDL 管理
- `/admin/resources`：资源管理
- `/admin/community`：社区内容管理
- `/admin/users`：用户管理

## 7. 导航结构建议

主导航：首页、学校与专业、选校推荐、申请管理、资源库、社区。

登录后侧边栏：工作台、背景档案、选校推荐、申请清单、任务时间线、文书简历、推荐信、Offer 对比、账号设置。
